# PDE Numerical Simulation Project

## Description

This project contains numerical simulations and computational studies for various partial differential equations (PDEs), focusing on soliton solutions, rogue waves, and Gaussian-type solutions.

## Project Structure

The repository is organized into two main research areas:

### 1. Bad-Boussinesq Equation

This directory contains numerical simulations and analysis of the Bad-Boussinesq equation, studying various solution types:

- **3rd-soliton/**: Third-order soliton solutions
- **Gauss/**: Gaussian-type solutions
- **rogue/**: Rogue wave phenomena
- **soliton-rogue/**: Interactions between solitons and rogue waves

### 2. Bad-JM Equation

This directory contains studies of the Bad-JM equation with different boundary and initial conditions:

- **jmcase1/**: First initial boundary condition case
- **jmcase2/**: Second initial boundary condition case
- **jm-gauss/**: Gaussian initial condition studies
- **jm-inverse-case2/**: Inverse problem studies for case 2

## Getting Started

### Prerequisites

- Python 3.x
- Required numerical computing libraries (numpy, scipy, matplotlib, etc.)

### Usage

Navigate to the specific study directory of interest and run the corresponding simulation scripts. Each subdirectory contains specific numerical experiments and visualization code for the respective PDE problems.

## Mathematical Background

### Bad-Boussinesq Equation

The Bad-Boussinesq equation is a nonlinear dispersive partial differential equation that models wave propagation in various physical systems. It exhibits interesting phenomena including:
- Soliton solutions
- Rogue wave formation
- Wave interactions

### Bad-JM Equation

The Bad-JM equation represents another class of nonlinear PDEs with applications in fluid dynamics and wave theory. This project investigates:
- Different initial boundary conditions
- Gaussian pulse propagation
- Inverse problems

## Research Focus

- **Soliton Dynamics**: Study of solitary wave solutions and their stability
- **Rogue Waves**: Analysis of extreme wave formation and evolution
- **Numerical Methods**: Implementation of computational schemes for PDE solving
- **Wave Interactions**: Investigation of nonlinear wave collision and superposition
